# 如何編譯
## 必要項
* 請安裝'electron-packager'
## 支緩的CPU架構&開始編譯
### Windows
#### ia-32(32位元)
```
$ npm run build_win32 
```
#### x86-64(64位元)
```
$ npm run build_win64
```
### linux
#### x86-64
```
$ npm run build_linux64
``` 

#### ARM32
```
$ npm run build_linuxarm32
```

#### ARM64
```
$ npm run build_linuxarm64
```
